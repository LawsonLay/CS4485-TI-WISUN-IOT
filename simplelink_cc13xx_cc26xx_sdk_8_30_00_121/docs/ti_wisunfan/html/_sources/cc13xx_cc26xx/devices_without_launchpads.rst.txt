Devices without LaunchPads
--------------------------

Most of SimpleLink devices are released along with a LaunchPad for evaluation
purposes. The SimpleLink SDK provides software examples for these LaunchPads.
However, there are some devices for which there is not released a LaunchPad.

The table below shows how to start developing on devices without  dedicated
evaluation board.

1. Start with a software example from the folder in the "Recommended SW example
   folder" column.
2. You need an evaluation board from the "Recommended LaunchPad" column.
3. Purchase a copy of your selected device from ti.com.
4. Solder your device onto the LaunchPad.
5. Follow the steps in the guide listed in the "Guide" column to migrate the SW
   example to your device.
